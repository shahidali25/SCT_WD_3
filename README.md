# SCT_WD_3
An interactive Quiz Game Application built using HTML, CSS, and JavaScript that displays questions, collects user answers, and shows the final score.
Quiz Game Application

This is a simple Quiz Game built using HTML, CSS, and JavaScript.
The application shows multiple-choice questions. Users select an answer and move to the next question. At the end of the quiz, the total score is displayed.

Features

- Multiple choice quiz questions
- User selects one answer
- Score calculation
- Final result displayed
- Restart quiz option

Technologies Used

- HTML
- CSS
- JavaScript

Project Files

- index.html
- style.css
- script.js

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in your browser.
